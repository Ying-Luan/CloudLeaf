# Chrome 扩展隐私政策

([English Version](PRIVACY.md))

**生效日期:** 2025-12-31

**CloudLeaf** (以下简称“本扩展”) 致力于保护您的隐私。本文档说明了当您使用本扩展时，我们如何收集、使用和保护您的信息。

### 1. 我们收集的信息

*   **非个人信息:** 本扩展不收集除启用其功能所需之外的任何非个人信息。如果将来收集此类信息，将在更新的隐私政策中明确说明。

*   **个人信息:** 本扩展使用以下个人信息:
    *   **书签:** 本扩展读取和修改您的浏览器书签以提供同步功能。此数据仅在本地处理或安全地传输到您配置的云存储提供商。
    *   **云存储凭据:** 我们存储您的 GitHub Token 或 WebDAV 凭据（用户名/密码）以在您选择的云提供商处进行身份验证。这些凭据安全地存储在您的浏览器本地存储中，绝不会发送到我们的服务器。

### 2. 我们如何使用您的信息

收集的信息仅用于:
*   在您的浏览器和您配置的云存储（GitHub Gist 或 WebDAV）之间同步您的书签。
*   协助在您选择的云提供商处进行身份验证。

我们不会与第三方共享、出售或出租您的个人信息。所有数据传输均直接发生在您的浏览器和您配置的云提供商之间。

### 3. 数据安全

我们实施合理的安全措施来保护您的信息。您的凭据存储在浏览器的本地，仅用于与您配置的云服务进行直接通信。

### 4. 数据保留

我们不在我们的服务器上保留您的任何个人数据。您的数据保留在您的浏览器和您的个人云存储中。

### 5. 用户权利

您有权:
*   随时删除扩展和所有本地存储的数据。
*   直接通过您的云提供商（GitHub 或 WebDAV 服务）管理您的云数据。

要行使这些权利，请联系我们: [3433800035@qq.com](mailto:3433800035@qq.com)。

### 6. 本隐私政策的更改

我们可能会不时更新本隐私政策。任何更改都将在此页面上发布，并附带更新的生效日期。我们鼓励您定期查看本政策。
